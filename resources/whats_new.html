Welcome to DataPlotStudio

A data analytics and plotting tool all in one.

Latest Features:

  • Text manipulation: Perform text manipulation on data containing strings. Remove whitespace and change casings, title, and capitalize in order to standardize text data.
  • Calculate column: Create a new columns and calculate their value based on other columns in the dataset. Supports arithmetic, comparison and logical operations.
  • Sorting: Make permanent sorting of data before exporting. Also added sorting as a tool in the right hand side of the data tab.
  • Interpolation: Use linear interpolation and time interpolation to fill missing values in your data.
