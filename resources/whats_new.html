Welcome to DataPlotStudio

A data analytics and plotting tool all in one.

Latest Features:

  • CRS Support: Coordinate Reference System Transformation for geospatial plots.
  • Basemap support: Adding basemap tiles from Esri, OpenStreetMap, CartoDB as background maps
  • Picker tools: Allows for clicking on elements in plot canvas and be moved to appropriate settings for that element
  • TwinX Support: Allows a secondary y axis to be plotted with its own scale
  • Export plot: Added a button that gets relevant settings before exporting the plot. Is easier to find that the embedded tool from matplotlib that is hidden in the navigatortoolbar of the canvas
  • Quick Filter: Added a quick filter tool that can be used before plotting to write queries that filter the data based on an expression.
  • Theme creator: Add default and custom themes to the plot. These are predefined JSON files or custom themes that can be loaded.
  • Edit Themes: Edit custom themes or create copies of the default themes using the JSON editor.

Need help? Check the Help menu for documentation and tutorials.
