Welcome to DataPlotStudio

A data analytics and plotting tool all in one.

Latest Features:

  • Table Search Functionality: Search for values in the data table and highlight them. Useful for editing data.
  • Database Connectivity: DuckDB database connection added.
  • Drag-and-drop files: Drag files from explorer directly into the datatable to view it.
  • Better Filtering: Filter multiple times on a single dataset to get the exact data you need.
