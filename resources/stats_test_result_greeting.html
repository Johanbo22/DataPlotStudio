How to run an analysis

  1. Navigate to the Data Table tab.
  2. Right-click anywhere on the table to open the context menu.
  3. Select Table Settings...
  4. Under the Behavior option, set Selection Behavior to Select columns.
  5. Return to the table and select exactly two numeric columns.
  6. Right-click your selection.
  7. Select Run Statistical Test... and choose your desired test.
Your test results, p-values, and statistical interpretations will appear here.
